# Daily Grind Server

A basic Express.js server created for the Per Scholas Node and Express module.

## Features
- Runs a local server using Express
- Displays a welcome message in the browser
- Uses Node.js and npm

## Technologies Used
- Node.js
- Express.js

## How to Run
1. Install dependencies
npm install

2. Start the server
node server.js

3. Open in browser
http://localhost:3000
